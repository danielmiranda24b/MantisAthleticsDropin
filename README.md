A pre-launch "coming soon" landing page for **Mantis Athletics** — a sports and athletics brand offering drop-in classes and training sessions. The page collects email waitlist signups and builds anticipation for the brand's launch.

---

## Features

- **Email Waitlist Form** — Collect interested users before launch
- **Service Preview** — Overview of upcoming drop-in class offerings
- **Social Media Links** — Instagram and other channels
- **Coming Soon Countdown / Teaser** — Build pre-launch buzz
- **Premium Typography** — Playfair Display (serif) + Roboto via Google Fonts
- **Dark Gradient Design** — Professional dark blue/teal aesthetic

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)
